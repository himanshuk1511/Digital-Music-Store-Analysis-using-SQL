# Digital-Music-Store-Analysis-using-SQL
–Performed SQL-based analysis on a relational store database (11 tables, 50K+ records) using JOINs, GROUP BY,
CTEs, window functions & subqueries to extract insights on customer behavior, revenue trends, & sales performance.
– Built 20+ optimized SQL queries to identify top customers, high-revenue cities, and popular genres, leveraging
multi-table joins and aggregations to support data-driven business decisions and targeted strategies.
